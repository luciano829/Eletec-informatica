Este repositório contém o código-fonte do site institucional da **Eletec Informática**, focado em apresentar serviços especializados para residências, comércios e indústrias.

---

## 🛠️ Serviços Oferecidos

*   **Segurança Eletrônica (CFTV):** Instalação de câmeras IP/Analógicas com acesso remoto.
*   **Elétrica Profissional:** Instalações prediais, comerciais e industriais (NBR 5410).
*   **Cabeamento Estruturado:** Organização de racks, redes de dados e Wi-Fi de alta performance.
*   **Interfonia:** Sistemas de comunicação e vídeo-porteiros inteligentes.

## 🚀 Tecnologias Utilizadas no Site

*   **HTML5 / CSS3:** Estrutura e estilização responsiva.
*   **Font Awesome:** Ícones vetoriais para serviços.
*   **Google Fonts:** Tipografia moderna (Roboto).
*   **JavaScript:** Carrossel de serviços e interatividade.
